# Простой Калькулятор на JavaScript

Это простой калькулятор на JavaScript, который выполняет базовые арифметические операции: сложение, вычитание, умножение и деление. Проект создан для обучения и практики работы с HTML, CSS и JavaScript.

## Функционал
- Сложение, вычитание, умножение и деление.
- Проверка на деление на ноль.
- Валидация ввода — проверяет, что оба поля заполнены числами.
- Отображение результата прямо на странице.

## Технологии
- HTML5
- CSS3
- JavaScript (Vanilla JS)
  

Как использовать

1 Введите первое число в поле «Первое число».

2 Выберите арифметическую операцию (+, -, *, /).

3 Введите второе число в поле «Второе число».

4 Нажмите кнопку «Вычислить».

5 Результат отобразится ниже кнопки
